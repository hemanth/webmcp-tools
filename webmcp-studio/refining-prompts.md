# Refining Prompts for WebMCP Studio

Use these prompts in the Antigravity IDE to iteratively improve the WebMCP tools
generated by the starter prompt. Copy and paste the relevant block into the
chat.

## Add missing interaction

**Use this if:** The agent missed a specific feature you know exists (e.g., a
"Sort by" dropdown).

```
I want to expose the "[INSERT FEATURE NAME]" functionality found on the page.

**Task:** Analyze the UI element for this feature (e.g., dropdown, button, list).

**Action:** Create a new tool named [INSERT_VERB_NOUN].

**Schema:** Define input parameters that match the visible options in the UI (use Enums if applicable).

**Output:** Append this new tool to webmcp-tools.js and webmcp-forms.html.
```

## Fix incorrect parameter

**Use this if:** The agent guessed a parameter name or type wrong.

```
The tool `[TOOL_NAME]` has an incorrect parameter `[WRONG_PARAM]`.

**Correction:** The actual URL parameter used by the site is `[CORRECT_PARAM]`. This parameters is used for `[SOMETHING]` and expects a `[TYPE]`.

**Action:** Update `webmcp-tools.js` and `webmcp-forms.html` to reflect this.
```
